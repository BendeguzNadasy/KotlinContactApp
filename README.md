# Contact App - Fullstack Application
This is a fullstack application designed to manage contacts.

The project is divided into two main parts:

## Project Structure

### 1. [Frontend (Client)](./contact-fe)
* **Tech Stack:** React, Vite, Bootstrap, Axios.
* **Description:** The user interface for listing, adding, and editing contacts.
* **[Read Frontend Documentation](./contact-fe/README.md)**

### 2. [Backend (Server)](./contact-be)
* **Tech Stack:** Kotlin, Spring Boot, REST API.
* **Description:** The server-side logic handling data persistence, file uploads, and business logic.
* **[Read Backend Documentation](./contact-be/README.md)**

---

## Quick Start Guide

To run the full application locally:

1.  **Start the Backend:** Navigate to the backend folder and run the Spring Boot application (port `8080`).
2.  **Start the Frontend:** Navigate to the frontend folder, install dependencies (`npm install`), and run the dev server (`npm run dev`).
